üìã Alternative Installation Methods

Method 1: Import from URL

  1. Open Tampermonkey Dashboard
  2. Go to "Utilities" tab
  3. Under "Import from URL", paste: https://raw.githubusercontent.com/withLinda/LinkedIn-profile-scraper-lite/main/build/linkedin-scraper.user.js
  4. Click "Import" and then "Install"

Method 2: Manual Installation

  1. Open Tampermonkey Dashboard
  2. Click the "+" tab to create a new script
  3. Copy the entire script from here
  4. Replace the default content with the copied script
  5. Press Ctrl+S (or Cmd+S on Mac) to save
